# LinkTracker

**LinkTracker** – Telegram-бот, который отслеживает изменения на веб-страницах и оперативно информирует пользователя о них.

## Запуск проекта

1. Создайте файл `.env` в корне проекта и добавьте в него токен вашего Telegram-бота:
   ```env
   APP_TELEGRAM_TOKEN=ваш_токен_здесь
   ```

2. Запустите бота командой:
   ```bash
   go run services/bot/cmd/bot/main.go
   ```
